# P5JS-Pong-Game
A classic Pong videogame created with Javascript and P5JS library from https://processing.org/

Instructions:

Press spacebar to start/resume the game

*Player 1:*
 
Move Up: W
 
Move Down: S


*Player 2:*

Move Up: Up arrow

Move Down: Down arrow
